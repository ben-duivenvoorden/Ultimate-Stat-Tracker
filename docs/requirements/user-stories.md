# User Stories
## Ultimate Stat Tracker

**Version:** 0.1 (Phase 0 — Requirements Gathering)
**Last Updated:** 2026-04-17
**Status:** 🟡 In Progress

---

## Sideline Recorder

- As a sideline recorder, I want to load a pre-built roster so I don't have to enter player names manually before a game.
- As a sideline recorder, I want to select which players are on the field at the start of each point so the app knows who is active.
- As a sideline recorder, I want to see only valid next actions after each event so I can't accidentally record an impossible sequence.
- As a sideline recorder, I want to record a stat in as few taps as possible so I don't miss live action.
- As a sideline recorder, I want to undo or correct my last entry quickly so mistakes don't corrupt the data.
- As a sideline recorder, I want the screen to always show me which team is attacking in which direction so I stay oriented during the game.
- As a sideline recorder, I want to see the current score at all times so I know the game state.
- As a sideline recorder, I want the app to automatically handle direction flips after each point and at half time so I don't have to manage that manually.

## Team Admin

- As a team admin, I want to create and manage a roster via the backend so players are ready to select before any game.
- As a team admin, I want to assign jersey numbers to players so the recorder can identify players by number quickly.

## Analyst

- As an analyst, I want to export clean game data so I can import it into my analysis package without manual cleanup.
- As an analyst, I want each event to reference a specific player, event type, and sequence position so I can reconstruct the full flow of any point.
